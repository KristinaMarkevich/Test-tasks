Использование техники тестирования "Таблица принятия решений"

В дождливую погоду ты надеваешь дождевик, когда идет снег - надеваешь пальто, когда светит солнце - надеваешь солнечные очки, когда нет дождя и снега, ты носишь плащ. Считать, что может идти только один вид осадков (дождь и снег не идут одновременно).

| Условие   | Тест1 | Тест2 | Тест3 | Тест4 | Тест5 | Тест6 |
|-----------|-------|-------|-------|-------|-------|-------|
| Дожь      | Есть  | Есть  | Нет   | Нет   | Нет   | Нет   |
| Снег      | Нет   | Нет   | Есть  | Есть  | Нет   | Нет   |
| Солнце    | Нет   | Есть  | Нет   | Есть  | Есть  | Нет   |
| **Результат** |       |       |       |       |       |       |
| Дождевик  | Да    | Да    | -     | -     | -     | -     |
| Пальто    | -     | -     | Да    | Да    | -     | -     |
| Плащ      | -     | -     | -     | -     | Да    | Да    |
| Очки      | -     | Да    | -     | Да    | Да    | -     |
